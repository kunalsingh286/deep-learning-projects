# 🚀 Deep Learning Projects Repository

This repository contains a collection of end-to-end Deep Learning implementations covering core supervised learning paradigms including classification and regression tasks. Each project is implemented in Jupyter Notebook with a clear workflow: data preprocessing → model building → training → evaluation.

---

## 📂 Projects Included

### 🔹 Classification Tasks

1. Binary Classification  
File: binary_classification.ipynb  
- Predicts between two classes (0/1)  
- Uses Sigmoid activation and Binary Cross Entropy loss  
- Example: Spam detection, disease prediction  

2. Multi-Class Classification  
File: multi_class_classification.ipynb  
- Classifies inputs into multiple categories  
- Uses Softmax activation and Categorical Cross Entropy  
- Example: Image classification, digit recognition  

3. Multi-Label Classification  
File: multi_label_classification.ipynb  
- Assigns multiple labels to a single input  
- Uses Sigmoid activation (per label) and Binary Cross Entropy  
- Example: Movie genre tagging, text categorization  

---

### 🔹 Regression Tasks

4. Single Output Regression  
File: single_output_regression.ipynb  
- Predicts a single continuous value  
- Uses Linear output layer and Mean Squared Error (MSE)  
- Example: House price prediction, sales forecasting  

5. Multi-Output Regression  
File: multi_output_regression.ipynb  
- Predicts multiple continuous outputs  
- Uses multi-neuron output layer and MSE loss  
- Example: Multi-dimensional forecasting, stock prediction
